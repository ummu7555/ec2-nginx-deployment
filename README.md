# EC2 NGINX Deployment

## Project Overview
This project demonstrates deployment of a static website using NGINX web server on an AWS EC2 instance.

## Technologies Used
- AWS EC2
- Linux
- NGINX
- Git & GitHub

## Steps Performed
1. Created EC2 instance in AWS
2. Connected to server using SSH
3. Installed NGINX web server
4. Cloned GitHub repository
5. Moved website files to NGINX root directory
6. Started and enabled NGINX service

## Result
Static website successfully hosted using NGINX on EC2.

## Learning Outcome
- EC2 instance setup
- Linux command usage
- NGINX configuration
- GitHub deployment workflow
